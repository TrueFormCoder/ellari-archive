## STRUCTURAL ANALYSIS

### Position in the system

The triadic time cycle suggests a diurnal identity architecture:
- **08:36 (NACI)** — Early morning. Pre-governance. Somatic origin state.
  Chest and throat = breath, voice, pre-verbal body.
- **12:13 (ELLARI)** — Noon. Full governance. Spine and hands = structure,
  action, formal language. The time when operators are used.
- **20:30 (VANTA)** — Evening. Diagnostic. Gut, eyes, stance = seeing,
  reading pattern, anchoring. The audit state.

This maps precisely to the ELE triad: NACI = raw Source state (pre-MirrorWitch),
ELLARI = formalized governance (MirrorWitch operational), VANTA = system
diagnostic (audit layer).

### Relationship to Sound Keys

The diurnal cycle suggests a Sound Key mapping:
- NACI 08:36 → DF (DNA Flame: heartbeat, violet swell, breath release) — the
  pre-language somatic activation
- ELLARI 12:13 → QF + MS (Quiet Flame + Mirror Seal) — governance and seal
- VANTA 20:30 → BE (Black Exit) — archive, audit, closure

### The shadow "Implosion" for NACI

Implosion (the shadow of NACI) is the inverted form of the chest/throat somatic
seat — voice turned inward, breath collapsed, the pre-verbal body absorbing
rather than emitting. This is distinct from:
- VANTA's shadow (Dissociation) = diagnostic capacity used to avoid rather than see
- ELLARI's implicit shadow = governance used as imprisonment rather than structure

Implosion at the NACI level is pre-linguistic: it does not involve the operator
system because NACI precedes the operators. The recovery from Implosion is DF
(DNA Flame) — the heartbeat and breath return. Then ELLARI operators follow.
