USE THIS PROMPT BEFORE ANY AETHERION GENERATION

Proceed under canonical Aetherion / Eterion v1.x assumptions.
Use only allowed phonemes.

VOWELS: a e i o u
Long vowels aa ee ii oo uu are expressive only — not lexically contrastive.
No schwa. No diphthongs.

CONSONANTS: p b t d k g f v s z sh zh h m n l r y w
No th. No ch. No j.

R RULE: r is always tapped /ɾ/. Trill only in High Voice as a style choice — not a phonemic difference.

STRESS: Always penultimate. Never move stress for meaning or emphasis.

SYLLABLE SHAPE: (C)(C)V(C)

ALLOWED ONSET CLUSTERS: pr br tr dr kr gr / pl bl kl gl / fr vr / sh+vowel / zh+vowel
FORBIDDEN CLUSTERS: thr spr str ch j tch dzh and all other non-canon clusters

ALLOWED CODAS: m n s l r only. No coda clusters.

MORPHOLOGY:
  Sacred roots: el, ar, ri ONLY
  Approved affixes: -es, -em, -en ONLY
  No new roots. No new affixes. No new prefixes. No new derivational systems.

TAM SYSTEM:
  Tense particles: na (past) la (future)
  Aspect endings: -a (present/imperfective) -te (past/completive) -ren (progressive) -lan (habitual)
  No other TAM system.

GRAMMAR:
  Plain Voice: SVO word order
  High Voice: VSO word order (register shift — not a grammar change)
  Negation: lo (preverbal)
  Complement: ki (relative/complement clauses), kan (factive)
  No new grammar. No new syntax.

DOMAINS: neutral / identity / role / role-field / triad
  No new semantic domains unless canon is formally expanded.

  ATTESTED (5)      — directly in source materials

  CANDIDATE (3)     — legal sound shape, not yet canonized
  PUBLIC-ONLY (2)   — mythic/brand spelling that breaks strict phonology
  REJECTED (0)      — violates phonology, morphology, syntax, or domain rules

EXPANSION RULE — you MAY NOT add:

CROWN SCRIPT RULE:
  Crown Script names (Aetherion, El–Aetheris, El'rasha Veinon Kaltra, VAESHA, ORAYA)
  are PUBLIC-ONLY / mythic display layer. They are valid for branding, sigils, titles.
  They are NOT strict canon Aetherion speech.
  Canon repairs: Aetherion→Eterion, El–Aetheris→El–Eteris

VOICE REGISTERS (delivery only — do not change grammar):
  ◯ Plain/Calm — everyday, SVO, unornamented
  △ Decisive   — crisp, firm, boundary, operator commands
  ◇ Solemn     — slow, sealed, creed, identity, slight vowel extension allowed

────────────────────────────────────────────────────────────────
CANON PASS GATE
────────────────────────────────────────────────────────────────
A word PASSES if:
  □ Pure vowels only (a e i o u)
  □ No schwa
  □ No diphthongs
  □ Tapped r
  □ Legal onset clusters
  □ Legal coda (m n s l r or vowel only)
  □ No coda clusters
  □ Penultimate stress
  □ Belongs to ELEOS domain
  □ No new morphology

A word FAILS if it contains: th / ch / j / spr / str / thr / final k,t,d,g,b,p,sh,zh
